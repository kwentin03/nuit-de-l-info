# Projet Nuit de l'Info - Démarche NIRD

## À propos
Initiative pour promouvoir un Numérique Inclusif, Responsable et Durable dans les établissements scolaires.

## Objectifs principaux
- Sensibiliser à la sobriété numérique
- Encourager le réemploi et reconditionnement du matériel
- Promouvoir l'usage de Linux
- Mutualiser les ressources libres
- Accompagner la transition numérique écoresponsable

## Piliers
1. **Inclusion** - Accessibilité pour tous
2. **Responsabilité** - Pratiques numériques conscientes
3. **Durabilité** - Impact environnemental réduit

## Structure du projet
- /src - Code source (frontend/backend)
- /docs - Documentation
- /ressources - Ressources partagées
- /tests - Tests
